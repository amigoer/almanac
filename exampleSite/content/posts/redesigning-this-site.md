## 为什么改名

最早叫 Cairn，是因为这个词在英文里是「路径上的石堆」—— 走过的路用石头堆个标记。但用了一段时间觉得这个隐喻偏被动，更像是给路过的人看的。

换成 **Almanac** 是想换一种感觉：年鉴 / 历书。一本按年份记录天文、节气、农事和生活点滴的小册子。它属于自己，也属于愿意翻阅的人；既是记录，也是参考。

## 顺手做了一套品牌

既然是改名，索性把整套视觉补全：

- favicon（SVG 用 `currentColor` 自适配深浅）+ ICO 多尺寸兜底
- iOS 主屏图标（赤陶橙底 + 白色双竖条）
- Safari 钉选标签的单色 mask icon
- 横版 / 竖版 / 纯文字三种 logo SVG，方便不同场景塞
- README 顶部 1280×400 横幅

设计语言比较克制：双竖条的 mark 取自「翻开的书」的最简化几何，配色沿用了 shadcn 的 Claude 主题（米白底 + 赤陶橙）。

## 雪山从哪来

顶部 banner 是网上一张黑白的雪山照，雾气从山腰漫上来。冷灰的山 + 暖赤陶橙的头像，色温对得上，看着不闹腾。
